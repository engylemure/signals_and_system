# signals_and_system
work on signals and systems involving audio signal processing using the Advanced Linux Sound Architecture (ALSA) library.


## Initial Setup for ALSA
Execute this commands
```
chmod +x install.sh && ./install.sh
```

### Also for remeber
```
gcc -lasound main.c
```
